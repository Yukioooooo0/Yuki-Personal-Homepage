这是一个示例文章，展示博客文章页面的基本结构。你可以在这里写技术笔记、读书笔记、或随笔。

示例代码

# 初始化仓库
git init
git add .
git commit -m "init"

要添加新文章：在 public/posts/ 中新增一个 HTML 文件，并在 public/blog/index.html 中添加链接即可。

如果希望支持 Markdown，可以使用静态站点生成器（如 Hugo、Jekyll）或在构建时把 Markdown 转为 HTML。
